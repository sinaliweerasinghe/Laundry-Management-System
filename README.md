# 🧺 Laundry Management System

## 📖 Overview

The Laundry Management System is a web-based application developed to streamline laundry service operations. This module focuses on **Categorization & Pricing Management**, allowing administrators to manage laundry item categories and their corresponding service prices efficiently.

## ✨ Features

- Add, edit, and delete laundry categories
- Manage pricing for different laundry services
- Categorize clothing and laundry items
- User-friendly admin interface
- Input validation and error handling
- Dynamic database integration
